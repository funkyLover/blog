##### [2019-04-09]git仓库迁移

摘录自 https://segmentfault.com/a/1190000012821833

``` bash
# 设置远程仓库地址
git remote set-url origin remote_git_address

# 推送到远程
git push origin --all # 推送所有分支

git push --tags # 推送标签
```
